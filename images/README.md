## 图标建议

1. 所有图标建议使用 PNG 格式，支持透明背景
2. tabBar 图标需要准备选中和未选中两种状态
3. NPC 图片建议使用你设计稿中的漫画风格黑白插画
4. 可以使用 iconfont 或 SF Symbols 获取基础图标

## 快速开始

如果你暂时没有图标资源，可以：

1. 先创建空的占位图片让项目能跑起来
2. 使用在线图标库（如 iconfont.cn）下载基础图标
3. 后续再替换为设计稿中的正式图标
